# Frontend - Large User Directory

## Overview

React application with TypeScript for displaying 10 million users efficiently using virtual scrolling.

## Features

- ✅ Virtual scrolling with react-window
- ✅ Infinite scroll loading
- ✅ Alphabet navigation menu
- ✅ Search with debouncing
- ✅ Responsive design
- ✅ TypeScript for type safety

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Starts the Vite development server (usually at http://localhost:5173).

## Production Build

```bash
npm run build
npm run preview
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:3000/api
```

## Technologies

- React 18
- TypeScript
- react-window (virtual scrolling)
- Axios (HTTP client)
- Vite (build tool)
